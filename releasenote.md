# Release Notes

## Version 1.1.8

### New Features
- **Audio Equalizer**: Added a 10-band equalizer with presets and persistent settings.
- **Redesigned Title Bar**:
    - Integrated search bar and main menu into the title bar.
    - Added breadcrumb navigation and search result counts.
    - Added maximizing state indicators, tooltips, and improved animations.@Manga-Knights 
- **Plugin Management**:
    - Added "Reinstall" option for plugins.
    - Added confirmation dialog for plugin auto-updates.

### Improvements
- **Native Feel**: Disabled global text selection for a more app-like experience.
- **Playlist Management**:
    - Unified context menu options in the sidebar (Play, Add to Queue, Rename, Change Cover, Delete).
    - Added confirmation dialog for playlist deletion.
- **Keybindings**: Added global keyboard shortcuts for better navigation.

### Bug Fixes
- **Plugin System**: Fixed issues with plugin update detection.
- **UI**: Fixed context menu overflow issues.
- **Lyrics**: Fixed lyrics seeking functionality in the Full Screen Player.
- **General**: Fixed fullscreen inconsistency. by @Manga-Knights
